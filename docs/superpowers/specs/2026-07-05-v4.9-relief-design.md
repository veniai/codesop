# v4.9 codesop 减负 spec

**日期**：2026-07-05
**主题**：codesop 减负（simple 出口 + 测试术语锚/措辞锁区分 + 去重降噪）
**作者**：claw + Claude（brainstorming）+ Workflow（6-finder 调研）+ codex（独立审）

## 背景

codesop 定位是「大模型的指引/锦上添花」，不是「限制」。Workflow 6-finder 调研 + codex 独立审 + Claude 对照，确认整体锦上添花为主，但有 3 处可减负（按 ROI 综合排序）：

1. simple/trivial 任务过重（缺跳 codex 出口）—— 运行时限制，直接影响大模型发挥
2. 测试锁措辞不锁行为（91 needle 卡 patch 改写）—— 演进限制
3. 多处重复声明（信号稀释 + SKILL.md 膨胀）

核心范式（spec-as-goal / 三 gate / 外部锚点 AND / diff 守护）不动。

## 目标

按综合排序减负，让 simple 任务少跑重流程、测试不再卡措辞、SKILL 信号密度提升。

## 范围

只改 codesop repo：`SKILL.md` / `config/codesop-router.md` / `patches/superpowers/brainstorming-SKILL.md` / `tests/` / `templates/system/AGENTS.md`（后者由 `setup --host claude` symlink 同步到 `~/.claude/CLAUDE.md`）。

**不含**：`~/.claude/CLAUDE.md:36` 全局 `1% skill` 规则（上游 superpowers 纪律，影响所有 skill，建议用户单独评估）。

### Section 4 — spec-gate 禁止降级 + 自动触发（codesop 自身漏洞修复，🥇brainstorming 过程发现）

> **背景**：brainstorming 交付 spec 时，AI（执行者）没自动 dispatch spec-gate 可视化，反而降级为文字 Layer 1/2 摘要 + 问用户要不要。这是 codesop 自身的执行漏洞——§8.7 B 写了机制，缺「禁止 AI 降级」硬约束。用户 v4.6 已提过同类问题（"必须我说了才做"），v4.6 改了内容/执行者，没改「禁止降级」。本 section 补这个洞。

#### 4.1 spec-gate 必 serve，禁止降级
- **完成条件**：`SKILL.md` §8.7 B 加硬约束「spec-gate 必 serve URL，**禁止降级为文字摘要/终端 Layer 1 输出**——serve URL 是 ready 的唯一信号，文字不算 ready」；`grep -F "禁止降级" SKILL.md` 命中
- **边界**：serve URL = ready 的语义不变（§8.7 B 已有），新增的是「AI 不得跳过 serve 直接给文字」硬约束
- **风险分级**：**low**

#### 4.2 brainstorming terminal state → codesop 自动 dispatch spec-gate
- **完成条件**：`brainstorming-SKILL.md` terminal state（交付 spec-gate）+ `SKILL.md` §8.7 B 明确「brainstorming 交付 spec 后，codesop **自动** dispatch spec-gate（不问用户要不要可视化）」；`grep -F "自动 dispatch spec-gate"` 在 brainstorming patch + SKILL 命中
- **边界**：用户在人审时仍可打回（保留人审权）；但 dispatch 本身不问——防 AI 把"问用户"当降级借口
- **风险分级**：**low**

#### 4.3 spec-gate B 四道约束放宽（关联，减负）
- **完成条件**：`SKILL.md` §8.7 B（含 :450 Layer 1 描述）的「独立 subagent + 不抄 spec + ≤5 行不用术语 + 强制 serve HTML」四道约束，**Layer 1 措辞放宽**（「优先白话可用术语」取代「不用术语」绝对禁令，**同步改 SKILL.md:450 的「不用术语」表述**——交叉检验发现否则 grep「优先白话」命中但旧措辞仍在 = 自相矛盾）+ serve 默认必做（不降级，呼应 4.1，保留）；`grep -F "优先白话" SKILL.md` 命中 且 `grep -F "不用文件名/章节号/术语" SKILL.md` = 0
- **边界**：独立 subagent（交叉检验核心）保留；diff 守护 / rubric 五项不动
- **风险分级**：**low**
